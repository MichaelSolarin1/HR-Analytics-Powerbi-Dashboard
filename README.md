# HR-Analytics-Powerbi-Dashboard
End-to-end HR Analytics Dashboard built in Power BI transforming raw workforce data into executive insights across engagement, performance, and retention. Showcases strong data modeling and business storytelling skills.
Executive Summary

This project delivers a fully interactive HR Analytics Dashboard built in Power BI to help leadership teams monitor workforce performance, reduce turnover, and optimize talent acquisition strategies.

Through structured data modeling and advanced DAX calculations, this dashboard transforms raw HR data into executive-level insights across engagement, performance, compensation, and recruitment efficiency.

The analysis reveals a 33% overall turnover rate, identifies 39 high-performing employees, and highlights recruitment channels driving the strongest talent acquisition results.

This solution demonstrates how HR data can be leveraged to support strategic workforce planning and improve retention outcomes.
